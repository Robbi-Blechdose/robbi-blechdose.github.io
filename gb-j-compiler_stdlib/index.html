GB-J Compiler: Standard Library Documentation

2019-11-05

0. Abbreviations

Abbreviation Description
Pos Position
VBK VRAM Bank
Ref. Reference to
N/o. Number of
BG Background

1. Standard Methods

Name Arguments Return Description
halt / / Puts the CPU into low-power modeuntil the next interrupt occurs.
waitVBlank / / Waits until a VBLank occurs. VRAM may be modified then.
enableInterrupts / / Enables interrupts.
disableInterrupts / / Disables interrupts.
softwareBreak / / Adds an ld b, b into the code on which BGB can break.
switchBank Bank Number / Switches Bank X to the given number. Possible values are 0-255 for MBC5, 1-127 for MBC3 and 1-127 excluding banks 32, 64 and 96 for MBC1.
copyDMARoutine / / Allows usage of sprites.
clearOAM / / Clears the sprite memory, otherwise garbage may appear on the screen.
dmaTransfer / / Transfers sprite data. Should always be called in VBlank.

2. Graphics

1.1. Sprites

Location: stdlib/Graphics/Sprites.asm

Name Arguments Return Description
setSpritePosition Sprite Index, X Pos, Y Pos / Sets the position of the sprite. Note: 0,0 is outside the screen area. The top-left corner of the screen is at 8, 16.
setSpriteTile Sprite Index, Tile Index / Sets the tile of the sprite.
setSpriteAttributes Sprite Index, Sprite Attribute / Sets the attributes of the sprite. See the [Sprite Attribute Table]() on how to create the attribute byte.
Sprite Attribute Table:
Bit Description
7 Sprite <-> Background Priority (0 = Sprite above Background)
6 Y Flip (0 = Normal)
5 X Flip (0 = Normal)
4 Palette Index DMG-Mode
3 VRAM-Bank of the tile
2-0 Palette Index CGB-Mode

1.2. Tiles, Background and Window

Location: stdlib/Graphics/Tiles.asm

Name Arguments Return Description
loadTiles VBK, Index, N/o. tiles, Ref. tile data / Loads tiles from the reference into VRAM. The index is the location in VRAM where the tiles are placed.
setBGTile VBK, Tile Index, X Pos, Y Pos / Places a single tile on the BG map at the given position. If VRAM Bank 1 is used, attributes for the tile are set instead.
setWinTile VBK, Tile Index, X Pos, Y Pos / Places a single tile on the window map at the given position. If VBK 1 is used, attributes for the tile are set instead.
loadBGMap VBK, Ref. map data, X Pos, Y Pos, X Size, Y Size / Loads a BG map and places it at the position and uses the given Size. If VBK 1 is used, attributes for the tiles are set instead.
loadWinMap VBK, Ref. map data, X Pos, Y Pos, X Size, Y Size / Loads a window map and places it at the position and uses the given Size. If VBK 1 is used, attributes for the tiles are set instead.
setBGScroll X Scroll, Y Scroll / Loads the BG scrolling registers with the given values.
setWinPosition X Pos, Y Pos / Places the window at the given position. Note: 0, 7 is the top-left corner of the screen.

1.3. LCD control

Location: stdlib/Graphics/LCD.asm

Name Arguments Return Description
lcdOff / / Turns off the LCD. It turns white for the user. VRAM access is now always possible.
lcdOn LCD On/Off, Window Map Address, BG On/off, BG Tile Data Address, BG Map Address, Sprite Size, Sprites On/Off, BG On/Off / Turns on the LCD with the given arguments.
loadPalettes Is Sprite Palette?, Ref. Palette Loads palettes into VRAM. CGB only.

2. Miscellaneous

2.1. Utils

Location: stdlib/Utils.asm

Name Arguments Return Description
readPad / / Reads the D-Pad and all buttons and saves the current status internally.
isButtonPressed Button Bitmask 1 = Button is pressed Checks if a button is pressed.
wasButtonPressed Button Bitmask 1 = Button was pressed Checks if a button was pressed in the last frame before the current one.
switchSpeed / / Switches between Single- and Double-Speed Mode. CGB only.
Button Bitmask Table
Bit Description
7 D-Pad ↓
6 D-Pad ↑
5 D-Pad ←
4 D-Pad →
3 Start
2 Select
1 B
0 A

2.2. GameBoy Printer support

Location: stdlib/GBPrinter.asm

Name Arguments Return Description
initPrinter / 1 = Connection OK Sends an init packet to the printer and waits for an answer.
transferData / / Sends the contents of the buffer to the printer in multiple frames. The buffer is the top left 20x18 tiles of the window layer.
getStatus / Printer Status Asks the printer for its status and returns the answer byte.
startPrint / / Attempts to start a print. Only works if enough data has been transferred beforehand.

2.3. Simple text display

Location: stdlib/Graphics/ScreenPrinter.asm
Disclaimer: Should only be used for testing purposes due to the amount of VRAM and CPU time used.

Name Arguments Return Description
loadCharset / / Loads the charset into VRAM. It contains letters A - Z, numbers 0 - 9 and the special characters ?,.!-.
print X Pos, Y Pos, Ref. Text / Prints text to the screen at the given position. Note: The position is in tiles here.
printNumber X Pos, Y Pos, char / Prints a number to the screen at the given position. Note: The position is in tiles here.
